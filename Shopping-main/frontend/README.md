# Frontend Application

## Description
This frontend application is built with React and JavaScript.  
It allows users to:
- View available products
- Add products to a cart
- Place an order via a REST API

## Architecture
The frontend follows a clean architecture:
- API layer for HTTP calls
- Pages for business views
- Reusable UI components
- Global cart state using Context API

## Technology Choices
- React with JavaScript for simplicity and clarity
- Axios for HTTP communication
- Context API for lightweight global state management
